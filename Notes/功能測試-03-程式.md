# 功能測試-03-程式

## Leibniz 級數近似 π

```python
pi = 0
sign = 1
for k in range(10000):
    pi += sign * 4 / (2*k + 1)
    sign *= -1
print("π ≈", pi)
```

## 牛頓法求 sqrt(2)

```python
x = 1.0
for _ in range(10):
    x = 0.5 * (x + 2 / x)
print("sqrt(2) ≈", x)
```

# 自由落體：y'' = -g

```python
g = 9.8
dt = 0.1
v = 0
y = 100
for t in range(20):
    v -= g * dt
    y += v * dt
    print(f"t={t*dt:.1f}s, y={y:.2f}m")
```

各程式區塊獨立，變數不連動
